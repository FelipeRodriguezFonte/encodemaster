# Changelog

All notable changes to the "EncodeMaster" extension will be documented in this file.

## [1.1.0] - 2024-12-25

### Added
- Hexdump command (xxd style) for debugging binary files
- Support for unsaved files (Untitled documents)
- Complete documentation with examples

### Features
- File to Hexadecimal conversion
- File to Base64 conversion
- File to Hexdump (xxd style)
- Hexadecimal to Binary file
- Hexadecimal to Base64
- Base64 to Binary file
- Base64 to Hexadecimal
- Selection to Hexadecimal
- Selection to Base64
- Hexadecimal to Text (selection)
- Hexadecimal to Base64 (selection)
- Base64 to Text (selection)
- Base64 to Hexadecimal (selection)

## [1.0.0] - 2024-12-25

### Added
- Initial release
- Basic binary/hex/base64 conversion functionality
- File-level conversions
- Selection-level conversions
